Design a modern AI-powered Final Year Project (FYP) Relevancy System dashboard UI in a clean SaaS style for a university platform. The design should look professional, futuristic, intelligent, and user-friendly while maintaining an academic environment.

STYLE & DESIGN LANGUAGE:
- Modern SaaS dashboard
- Minimal but premium
- Soft shadows
- Rounded cards and panels
- Smooth spacing and hierarchy
- Blue/purple AI-inspired theme
- Clean typography
- Subtle gradients
- Light mode with optional dark mode version
- Responsive desktop dashboard layout
- Professional university system aesthetic
- Add smooth hover interactions and micro-animations

MAIN GOAL:
Transform the current project management dashboard into an AI-powered project intelligence platform where AI features are visible everywhere, not hidden behind a single score.
